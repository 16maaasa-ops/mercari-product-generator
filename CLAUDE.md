# CLAUDE.md

このファイルは、リポジトリで作業する Claude Code (claude.ai/code) へのガイダンスを提供します。

## プロジェクト概要

商品写真と実寸を入力するだけで、メルカリ用の「商品名」と「説明文」を Claude API で生成する Next.js 製 Web アプリ。個人利用を前提とし、Vercel にデプロイする。

## 技術スタック

- **フレームワーク**: Next.js (App Router)
- **デプロイ先**: Vercel（無料枠）
- **AI**: Claude API（ビジョン対応の最新モデルを使用: `claude-opus-4-8` または `claude-sonnet-4-6`）
- **言語**: TypeScript

## 開発コマンド

```bash
npm run dev      # ローカル開発サーバー起動 (http://localhost:3000)
npm run build    # 本番ビルド
npm run lint     # ESLint
```

## アーキテクチャ

画面遷移なしのシングルページ構成。主要ファイルは以下。

- `app/page.tsx` — 入力フォーム（写真アップロード＋実寸）と結果表示を同一ページで管理
- `app/api/generate/route.ts` — Claude API を呼ぶ API ルート（APIキーをサーバー側で保持）
- `.env.local` — `ANTHROPIC_API_KEY` と `ACCESS_PASSWORD`（コミット禁止）

### API ルートの責務

`/api/generate` は以下を行う:
1. アクセスパスワードを検証（不正利用による API 料金増加を防ぐ簡易ガード）
2. アップロード画像を Base64 エンコードして Claude にビジョン付きで渡す
3. `productName`（40 文字以内）と `description` を JSON で返す

### 説明文フォーマット（F-05）

Claude へのプロンプトは以下の構成を厳守する。

**書き出し（固定）**
```
ご覧いただきありがとうございます！
即購入大歓迎です◎

値引き交渉、フォロー割も実施しております。ご質問等ありましたらコメント欄へお願いいたします(^^)
#古着屋36 ←他にも多数出品しておりますので、ご覧ください‼️
```

**本文ルール**
- 魅力ポイントを `✅` 先頭の箇条書きで 3〜5 個
- サイズの数字の後には必ず `cm` をつける
- サイズの直後に `※素人採寸のため、多少の誤差はご了承ください。` を入れる

**末尾（固定）**
- 大項目（ジャンル: アメカジ / Y2K / スポーツウェア / フォーマル など）
- 中項目（商品の種類: フーディー / ニット / ジャケット など）
- 小項目（デザイン・柄・状態などの詳細）
- 検索用キーワードを半角スペース区切りで並べる（`#` は使わない）

**プロンプトの制約**: 写真から確認できない実寸は絶対に捏造しない。不明な箇所は `[着丈〇〇cm]` のような空欄で返す。

## セキュリティ

- API キーは `ANTHROPIC_API_KEY` 環境変数で管理（サーバー側のみ、クライアントには非公開）
- API ルートで `ACCESS_PASSWORD` 環境変数を使った簡易パスワード認証を設け、第三者による無断利用を防ぐ

## UX の詳細

- 商品名: 編集可能なテキストエリア。リアルタイムの文字数カウンター（`XX/40`）を表示し、40 字超過時は赤で警告
- 説明文: 編集可能なテキストエリア
- 商品名・説明文それぞれに個別のコピーボタンを設置
- 生成中はローディングスピナーを表示（画像処理のため数秒〜十数秒かかる）
- 生成失敗時はエラーメッセージと「もう一度」ボタンを表示
